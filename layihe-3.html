Brauzerde Göstərmə

Artıq Kodlarımızı Brauzerlərdə Göstərək....

Əvvəcə This PC vəya Computer simgəsinə daxil olaq

Və qarşınıza Bu Pəncərə gələcək..

1. İlk Öncə View daxil oluruq

2. Sonra Options daxil oluruq

3. yenidən View daxil oluruq

4. və burada mütləq işarəli yeri Passiv edirik.

5. Və OK basaraq çıxırıq

Şəkili daha yaxşı görmək üçün şəkil'ə klik edin.

.html , .htm uzantılar

İlk öncə biz masaüstümüzə gəlib (Sağ Klik) edirik və kursoru (New) üstünə gətirik..

NotePad Açırıq..

və yaratdığımız NotePad'in adını hökmən (.html) və ya (.htm) uzantılı ad qoyuruq.. adətən html fayllarna (index) adı verilir yəni notePad"in adını (index.html/htm) siz (index) yerine istədiyiniz adı qoya bilərsiniz amma uzantını dəyişə bilmərsiniz...
Dəyişdikdən sonra sizə bir cavab mesajı gələcək siz ona (yes/да) təsdiqliyirsiz..
yənidən fayl'a (sağ klik) edirik və orda (Open with) kursoru gətirik.

NotePad seçirik) yəni biz burada `index.html` NotePad ilə aç dedik..

Əlaa artıq biz Kodlarımızı NotePad yəni(Code editor)'a yaza bilərik.. İndisə Gəlin Brauzerde göstərək..

Keçən dərsdə keçdiyimiz kimi Kodlamağa başlayarkən ilk öncə <html> tag ilə başlayır..

<html>
<head> <title>Sayt yaradırıq..</title> </head>

</html>

Və faylımızı (Save) etməliyikki Kodlarımızı Brauzerde görə bilək..

Save etdikdən sonra faylımızı açaq..(normal olaraq open with'ə yox)
